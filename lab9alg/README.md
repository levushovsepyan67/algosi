# Лабораторная работа 9: хеширование

В проекте реализован алгоритм хеширования `MD5` (собственная реализация, без `MessageDigest`).

## Запуск

```bash
mvn -q compile exec:java
```

После запуска введите строку, программа выведет её MD5 хеш в hex-формате.

## Полный прогон лабораторных тестов

```bash
mvn -q compile exec:java -Dexec.args="--run-lab"
```

Будут выполнены все 3 пункта задания:

- 1000 пар строк длины 128 с отличиями 1,2,4,8,16 и поиск максимальной общей последовательности в хешах.
- `N = 10^i`, `i=2..6`, строки длины 256 и проверка коллизий.
- 1000 замеров для размеров `64..8192` и вычисление скорости.

Результаты сохраняются в папку `lab9_results`:

- `diff_experiment.csv`
- `collision_experiment.csv`
- `speed_experiment.csv`
- `summary.md`
- `chart_diff_vs_common.png`
- `chart_size_vs_speed.png`
